Acordeón Informativo

Bootstrap es un framework CSS que proporciona componentes pre-diseñados y un sistema de grid responsive. Facilita el desarrollo de sitios web modernos y adaptables a diferentes dispositivos. Incluye componentes como navbars, cards, modales, y muchos más.

El diseño responsive permite que tu sitio web se adapte automáticamente a diferentes tamaños de pantalla. Esto mejora la experiencia del usuario en móviles, tablets y escritorio, además de beneficiar el SEO y reducir los costos de mantenimiento al tener una sola versión del sitio.

Entre las clases más útiles de Bootstrap encontramos: container y row para el sistema de grid, btn para botones, card para tarjetas, navbar para navegación, table para tablas, y clases utilitarias como text-center, mb-4, bg-primary, entre muchas otras.
